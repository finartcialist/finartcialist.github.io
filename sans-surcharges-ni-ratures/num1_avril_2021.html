TTType

L'environnement de développement Type and Trade and Type (TTType) a été conçu de façon à simuler un contexte de marché pour chacun des symboles utilisés dans un langage de programmation. Pour voir le projet: lien vers le projet TTType.

La personne souhaitant programmer dans TTType doit donc s'assurer d'avoir les caractères nécessaires dans son portefeuille afin de pouvoir exécuter son programme.

Bien que cette version de TTType aie été conçue pour un langage de programmation spécifique, le concept peut être adapté pour tout langage de programmation.

L'objectif de ce projet est de matérialiser une contrainte créative de marché. Au fur et à mesure que la tâche de programmation progresse, une tension se crée entre les possibilités offertes par le marché et celles offertes par la programmation...

TTType

The development environment Type and Trade and Type (TTType) simulates a market constraint for each symbol used in a programming langage. To access the project: link to the TTType project.

When programming in TTType, you need to make sure that your portfolio contains enough of each caracter to ensure a successful execution of the program.

While this version was made for a specific programming langage, the concept can be adapted to any programming language.

The goal of this project is to materialize a creative market constraint. As the programming task progresses, a tension is created between the opportunities created by the market and the ones afforded by programming...
